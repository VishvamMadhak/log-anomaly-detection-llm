# Realtime Log Anomaly Detection using TinyBERT

## Overview
This project builds a real-time log anomaly detection system using TinyBERT trained on the BlueGene/L (BGL) dataset.

## Features
- Log preprocessing pipeline
- Transformer-based anomaly detection
- Hyperparameter optimization using Optuna
- Real-time inference pipeline

## Tech Stack
Python, PyTorch, NLP, Optuna

## Results
- 98%+ classification accuracy
- High precision and recall for anomaly detection

## Dataset
BlueGene/L Supercomputer Log Dataset

## How to Run
pip install -r requirements.txt
python train.py
